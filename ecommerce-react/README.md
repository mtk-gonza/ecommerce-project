## 📦 Instalación y Uso

1. Clona el repositorio:
   ```bash
   git https://github.com/mtk-gonza/ecommerce-project.git
   cd ecommerce-react
   ```
2. Inicia el servidor de desarrollo:
   ```bash
   npm install
   npm run dev
   ```
3. Abre en tu navegador:
   http://localhost:5173
